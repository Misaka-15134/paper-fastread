# Release Checklist (Single Paper Lecture)

在交付HTML讲义前，逐项自检：

- [ ] 9章节完整且顺序正确：
  - [ ] 1. 论文概览卡片
  - [ ] 2. 研究逻辑与故事主线
  - [ ] 3. 引言导读
  - [ ] 4. 摘要精读
  - [ ] 5. Figure逐图精读与逻辑串联
  - [ ] 6. 核心方法学精读
  - [ ] 7. 方法学批判
  - [ ] 8. 讨论导读
  - [ ] 9. 课堂讨论题

- [ ] 每个Figure区块结构完整（顺序固定）：
  - [ ] `logic-box`
  - [ ] `caption-box`
  - [ ] `quote-box`
  - [ ] `steps-box`
  - [ ] `results-box`
  - [ ] `panel-details`

- [ ] 证据完整：
  - [ ] 每个Figure有Results连续3-5句英文原文
  - [ ] 图注翻译包含 `n=` / `P<` / `fold change` / `scale bar`

- [ ] 方法学深挖：
  - [ ] 第6章至少1-2个核心方法
  - [ ] 含Methods原文段落
  - [ ] 解释参数选择原因、严谨性、优势与局限

- [ ] 图片链接正确：
  - [ ] Elsevier使用`gr`主图ID
  - [ ] 未误用`fx`补图ID（除非明确需要补图）

- [ ] 样式完整：
  - [ ] 包含`.logic-box`样式
  - [ ] 包含`.quote-box`样式
