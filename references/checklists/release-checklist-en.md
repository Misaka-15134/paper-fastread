- [ ] Source setup confirmed (OpenAlex default; PubMed cross-check for biomedical papers)

- [ ] Results quotes contain 3+ consecutive sentences from main text
- [ ] Caption translation preserves `n`, `P`, `fold`, `scale bar` when present
- [ ] Section 6 includes Methods original text + design rationale analysis
- [ ] Figure links match correct figure indices and are accessible
